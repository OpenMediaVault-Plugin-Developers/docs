# TL;DR installing OMV 5 on Armbian Buster

1. Fresh install of armbian buster (minimal image works fine)
2. `apt update`
3. `apt upgrade`
4. `wget -O - https://github.com/OpenMediaVault-Plugin-Developers/installScript/raw/master/install | sudo bash`
5. `reboot`
6. Profit! :rocket:
